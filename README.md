## Installation Guide 

### Clone this repository
```bash
$ git clone https://github.com/SauravKumarMahato/Software_tree.git
```
### Go into the repository
```bash
$ cd Software_tree
```
### Install packages
```bash
$ pipenv install
```
### Run the app
```bash
$ python create_user_test.py

$ python powerTrade.py
```
